## ✨ Funcionalidades

- 🔍 **Buscador de gasolineras**: Consulta las gasolineras de España
- 📍 **Geolocalización**: Usa tu ubicación actual para buscar gasolineras cercanas
- 📏 **Radio personalizado**: Filtra gasolineras por distancia (a la redonda)
- 🛢️ **Precios en tiempo real**: Datos actualizados desde el API oficial del Minetur
- 📱 **Diseño responsivo**: Funciona en dispositivos móviles y escritorio

---

## 🚀 Cómo Usar

1. Abre el archivo `index.html` en tu navegador
2. Haz clic en **"Descargar Gasolineras"** para cargar los datos
3. O usa **"Usar mi ubicación"** para buscar gasolineras cerca de ti
4. Ajusta el **radio de búsqueda** en kilómetros (a la redonda)
5. Pulsa **"Buscar"** para ver los resultados

## 📊 Datos

Los datos provienen de la API oficial del Ministerio para la Transición Ecológica de España:

- URL: `https://sedeaplicaciones.minetur.gob.es/ServiciosRESTCarburantes/PreciosCarburantes/EstacionesTerrestres/`
- Actualización: Datos oficiales en tiempo real
